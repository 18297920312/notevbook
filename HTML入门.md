# HTML入门笔记1
1. HTML的发明者是Tim Berners-Lee
2. HTML的起手式必须按出！并按下TAB键就会出现
   ```html
   <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport"
        content="width=device-width, initial-scale=1.0, minimum-scale=1.0, maximum-scale=1.0, user-scalable=no">
    <title>Document</title>
</head>

<body>

</body>

</html>

3. 常用的章节标签有：
    * h1~h6 表示大标题
    * p     表示段落
    * section 
    * main 表示主要部分
    * article 表示文章内容
    * aside 
4. 全局属性：
    * class 类名 
    * hidden 隐藏该标签
    * id  除非万不得已不然一般不用id名
5. 常见的内容标签：
    * em 表示强调部分，主要是语气上的强调
    * strong 字体会加粗
    * a 链接的标签
    * pre 在此标签中空格、回车等特殊字符都会保留
    * code 此标签中一般显示代码
    * quote 此标签一般是引用
